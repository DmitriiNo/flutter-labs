# Мое приложение: MyNotes
MyNotes – мобильное приложение для быстрого создания и управления текстовыми заметками. Его простой интерфейс позволяет сосредоточиться на своих идеях и мыслях. Пользователь также может записать в заметках, например, свои планы на день, список покупок, какие-нибудь напоминания.
# Основные функции
•Создание новой заметки (ввод заголовка и текста)
•Просмотр списка всех заметок
•Редактирование заметок (изменение содержимого)
•Удаление заметок
•Сохранение заметок
•Поиск по заметкам

# Сценарии использования
1)Создание новой заметки
Пользователь открывает приложение, видит список всех заметок, нажимает на кнопку [+]. Далее открывается экран создания заметки, пользователь вводит заголовок и текст, нажимает сохранить, возвращается к списку, система показывает вверху списка новую заметку.
2)Просмотр и редактирование заметки
Пользователь нажимает на заметку в списке, которую он хочет отредактировать. Открывается заметка, пользователь меняет в ней либо заголовок, либо текст, затем нажимает сохранить. Дата изменения заметки меняется, пользователь возвращается к списку всех заметок.
3)Удаление заметки
Благодаря долгому нажатию на заметку открывается диалоговое окно подтверждения удаления. Пользователь нажимает кнопку «удалить». Список заметок обновляется, в нем уже не будет удаленной заметки.
4)Поиск заметок
Пользователь на главном экране в строке поиска вводит нужный ему текст. Система фильтрует заметки и находит нужную.
Доступные действия пользователя:
•Нажатие на заметку – открытие окна редактирования заметки
•Долгое нажатие на заметку – открытие диалогового окна удаления
•Нажатие кнопки {+} – создание новой заметки
•Ввод в поисковую строку – поиск заметок
•Нажатие "Сохранить" – сохранение изменения
•Нажатие "Назад" – автоматическое сохранение заметки и выход
